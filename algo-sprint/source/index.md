4. **难度**：Day 1～2 Easy 为主，Day 3～4 Easy+Medium，Day 5～6 Medium，Day 7 Medium+Hard。

## 单日学习流（四阶段）

每天分为 **热身 → 学习 → 实战 → 复盘** 四个阶段：

```
┌─ 阶段一：热身（15 min）
│   ├─ Day 2 起：从昨天 quiz.md 里挑一道做错/卡住的题，重做一遍（不看答案）
│   └─ 快速扫一眼 thinking.md 的「模式识别表」和「边界清单」
│
├─ 阶段二：学习（40～60 min）
│   ├─ datastructure.md → 当日小节
│   ├─ algorithm.md     → 当日小节
│   └─ combo.md         → 当日小节（含模式选用口诀）
│
├─ 阶段三：实战（90～120 min）
│   ├─ practice.md → 当日例题（先看题 + 渐进提示，自己写完再对答案）
│   └─ quiz.md     → 当日自测（限时，用 UMPIRE 流程）
│
└─ 阶段四：复盘（10 min）
    └─ 打开 review.md，按模板记录：掌握了什么、卡在哪步、漏了什么边界、明天要回顾什么
```

## 间隔复习机制

遗忘曲线告诉我们：学完第二天忘 70%。因此每天热身阶段都安排了回顾：

- **Day 2 热身**：重做 Day 1 的一道错题
- **Day 3 热身**：重做 Day 2 的一道错题
- **Day 4 热身**：重做 Day 2 或 3 的一道错题（间隔 2 天的内容）
- **Day 5 热身**：重做 Day 3 的一道错题（间隔 2 天）
- **Day 6 热身**：重做 Day 4 的一道错题
- **Day 7 热身**：重做 Day 5 的一道错题

依据 review.md 中「明天要回顾什么」来选择重做的题。
